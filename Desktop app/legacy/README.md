# Legacy Python prototype — niet voor productie

De oorspronkelijke Python/Dash-app staat hier uitsluitend als historische referentie.

Gebruik deze versie niet voor ontwerpbeslissingen of productieanalyses. Zij gebruikt onder andere
single-precision signaalarrays, permissieve parsing/decoding en heeft niet de integriteitsgaranties,
nanosecondenprovenance en strict/partial-status van CANalyser 2.0.

De enige productieversie is de .NET 8 WPF-app onder `src/`.
